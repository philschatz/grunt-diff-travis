This file contains the list of classes generated by Docbook and by CNX's custom Docbook+ conversion.

There are some auto-gen classes that can be used as well.
Gentext will generate text like "Figure 1.3: Octopus" from a language-based template that looks like this: "Figure %s: %t". In order to be able to style the number and title differently span tags are added around all of the text. It becomes:
<span class="cnx-gentext-figure cnx-gentext-autogen">Figure </span>
<span class="cnx-gentext-figure cnx-gentext-n">1.3</span>
<span class="cnx-gentext-figure cnx-gentext-autogen">: </span>
<span class="cnx-gentext-figure cnx-gentext-n">Octopus</span>

- ccap-sociology.less : sets style variables/classes used below and selectors for coloring features
- ccap-base.less : 
- docbook-and-online.less: given the classes defined above, generate rules for online and docbook
